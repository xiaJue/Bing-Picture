## PictureBing<br><br>
每天更新一张清晰的风景图片。
<br>数据来源：必应
<br>图片数据来源：https://bing.ioliu.cn/
<br>数据的获取:
<br>www.bing.com/HPImageArchive.aspx?format=js&idx=0&n=8
